# Problems Submission
---

## q0_MaxMeanSubArray

**Problem Type:** Array / Sliding Window  

**Description:**  
Finds the maximum average of any contiguous subarray of size `k` within an array of integers.

**Time Spent:** 15 minutes  

**Complexity Analysis:**  
- **Time Complexity:** O(n), where *n* is the length of the array    
- **Space Complexity:** O(1)

## q1_ReverseVowels

**Problem Type:** String / Two Pointers  

**Description:**  
Reverses only the vowels in a string while keeping all other characters in their original positions.

**Time Spent:** 20 minutes  

**Complexity Analysis:**  
- **Time Complexity:** O(n)  
- **Space Complexity:** O(n), where *n* is the length of the string

## q2_ZeroSumSubArrays

**Problem Type:** Array / Prefix Sum  

**Description:**  
Counts how many starting indices in an array have at least one contiguous subarray whose sum equals zero.

**Time Spent:** 20 minutes  

**Complexity Analysis:**  
- **Time Complexity:** O(n * n), where *n* is the number of elements in nums 
- **Space Complexity:** O(1)

## q3_BackspaceStringCompare

**Problem Type:** String / Stack  

**Description:**  
Compares two strings after processing `#` as a backspace character that removes the previous character if one exists.

**Time Spent:** 15 minutes  

**Complexity Analysis:**  
- **Time Complexity:** O(n + m)  
- **Space Complexity:** O(n + m), where *m* is the length of text1 and *n* is the length of text2

## q4_ShortestSubstring

**Problem Type:** String / Sliding Window  

**Description:**  
Finds the length of the shortest substring of `text1` that contains all characters of `text2`, including duplicates.

**Time Spent:** 30 minutes  

**Complexity Analysis:**  
- **Time Complexity:** O(n * n), where *n* is the length of text1
- **Space Complexity:** O(k), where *k* is the number of unique characters in `text2`

## q5_MissingInteger

**Problem Type:** Array / Set  

**Description:**  
Returns the smallest positive integer between `1` and `n` that is missing from the array. If none are missing, returns `"No missing number"`.

**Time Spent:** 10 minutes  

**Complexity Analysis:**  
- **Time Complexity:** O(n)  
- **Space Complexity:** O(n), where *n* is the size of the input array

## q7_MergeIntervals

**Problem Type:** Array / Sorting  

**Description:**  
Merges overlapping or touching intervals and returns a list of non-overlapping intervals sorted by start time.

**Time Spent:** 15 minutes  

**Complexity Analysis:**
